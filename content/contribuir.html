+++ title = "Contribuir" date = 2021-01-31 +++

Existen tres métodos distintos para enviar contribuciones para que sean incorporadas al sitio, por orden de preferencia según cuál tiene más probabilidades de ser incorporada:

  • Enviar una pull request en GitHub con la nueva página a añadir o modificar.
    Un responsable del sitio revisará tu propuesta y la aceptará, sugerirá cambios o la descartará.

  • Crear un issue en GitHub describiendo el nuevo contenido o modificación propuestos.
    Si la propuesta se considera de interés, algún programador podría implementarla.

  • Enviar un email a webmaster@bonitoperoinutil.com.
    El responsable de la web recibirá tu propuesta y la añadirá a la lista de issues si la considera de interés.
